# Тестовое задание 
## 1. Запуск docer compose
```
docker compose up --build
```
## 2. Swagger UI
Перейти на http://127.0.0.1:8000/docs 

Для краулера POST `/crawl` - параметры как в ТЗ

Для получения списка страниц GET `/pages` - возможен поиск как по url, так и по title в поле `q`

Для получения данных страницы (`HTML`, `URL`, `title`) GET `/pages/content` - по url.
